Full-stack web platform for renting mountain cottages in Serbia. Implements multi-role authentication, booking system, and responsive UI with dynamic maps.

## Features
- Multi-role authentication: Tourist, Owner, Administrator
- Cottage listing, search, sorting, and detailed views
- Booking system with multiple steps and payment integration
- Profile management for all user types
- Administrator dashboard for user and cottage management
- Responsive design for desktop and mobile

## Tech Stack
- **Frontend:** Angular, HTML, CSS, JavaScript/TypeScript
- **Backend:** Node.js
- **Database:** MongoDB
- **Tools:** VS Code, Postman
